## Dataset Overview
This dataset contains information about secondary school students, including demographic, social, and academic attributes. The main objective is to analyze factors affecting students' final grades.

## Number of Records
- 395 students  
- 33 attributes  

## Important Variables
- G1: First period grade  
- G2: Second period grade  
- G3: Final grade  
- studytime: Weekly study time  
- failures: Past class failures  
- absences: Number of school absences  

## Objective of Analysis
The purpose of this analysis is to explore the relationships between study habits, past academic performance, absences, and final grades using exploratory data analysis techniques.
